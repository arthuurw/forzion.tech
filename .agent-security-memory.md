## Decisões arquiteturais conscientes

### DC-001 — Escolha de provider LLM ✅ _Decidido em 2026-05-16_

**Decisão:** Claude Haiku-4.5 (`claude-haiku-4-5-20251001`) via Anthropic API.

**Justificativa:** qualidade em PT-BR superior a modelos locais; custo baixo (~$0.002/request com prompt caching); OCI VM atual sem GPU tornaria Ollama lento demais para UX aceitável.

**Custo estimado:**
- ~$3/mês em early stage (~50 req/dia)
- ~$30/mês em 100 users ativos (~500 req/dia)
- ~$150/mês em 500 users (~2.500 req/dia)

**Risco LGPD aceito conscientemente:** dados físicos de alunos processados em servidores Anthropic (EUA). Mitigações obrigatórias antes de ir para produção:
- Verificar e confirmar que API usage da Anthropic não é usada para treino de modelos (por padrão API não é, mas deve ser verificado nos termos vigentes)
- Documentar base legal LGPD para transferência internacional (Art. 33)
- Não incluir CPF, nome completo, ou dados de saúde clínica no contexto do LLM

**Credenciais:** via variável de ambiente / User Secrets. Não hardcoded. Não incluídas ainda (Sprint 1).
